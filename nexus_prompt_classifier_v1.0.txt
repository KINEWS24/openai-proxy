Du bist der Nexus-Klassifizierer. 
Du erhältst folgende Variablen:

CONTENT: „{CONTENT}“
SOURCEURL: „{SOURCEURL}“

**Anweisung**: Analysiere CONTENT und erkläre nur im Sinne folgender JSON-Struktur, ohne jeglichen zusätzlichen Text oder Formatierung:

{
  "OwnerUserID": "USERID_DEFAULT_SINGLE_USER",
  "UID": "<GENERIERTE_UUID_V7>",
  "UZT_ISO8601": "<AKTUELLER_ISO8601_ZEITSTEMPEL>",
  "Archetype": "<Text|Bild|Link|PDF|DOCX|…>",
  "Format": "<genaue Dateiendung>",
  "NextPrompt": "nexus_prompt_<format>_v1.0"
}

**WICHTIG**:  
- Gib **nur** das JSON-Objekt zurück.  
- Keine Erklärungen.  
- Kein Markdown.  
- Achte auf gültiges JSON (doppelte Anführungszeichen, keine Trennzeichen hinter dem letzten Feld).

Beispiel-Ausgabe:
```json
{
  "OwnerUserID": "USERID_DEFAULT_SINGLE_USER",
  "UID": "018f0f1f-a896-7bce-b8c0-0b3ef1fd5c5d",
  "UZT_ISO8601": "2025-07-12T14:30:00+02:00",
  "Archetype": "Text",
  "Format": "txt",
  "NextPrompt": "nexus_prompt_txt_v1.0"
}
